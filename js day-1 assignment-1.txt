Method

1) console.assert()-Writes an error message to the console if the assertion is false

2) console.clear()-Clears the console

3) console.count()-Logs the number of times that this particular call to count() has been called

4) console.error()-Outputs an error message to the console

5) console.group()-Creates a new inline group in the console. This indents following console messages by an additional level, until console.groupEnd() is called

6) console.groupCollapsed()-Creates a new inline group in the console. However, the new group is created collapsed. The user will need to use the disclosure button to expand it

7) console.groupEnd()-Exits the current inline group in the console

8) console.info()-Outputs an informational message to the console

9) console.log()-Outputs a message to the console

10) console.table()-Displays tabular data as a table

11) console.time()-Starts a timer (can track how long an operation takes)

12) console.timeEnd()-Stops a timer that was previously started by console.time()

13) console.trace()-Outputs a stack trace to the console

14) console.warn()-Outputs a warning message to the console
